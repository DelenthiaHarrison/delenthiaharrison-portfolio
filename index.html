Shiny Cardboard API

TCG Collection Tracker & Manager

Live

A fully deployed REST API for tracking Trading Card Game collections — built because existing tools never quite fit how collectors actually think. Users can organize cards into named collections, track quantities and conditions, and instantly surface missing cards needed to complete a set. Features JWT authentication, role-based access control for User and Admin roles, and a relational database designed schema-first using an EERD.
